## When using this device in published work

Add a line to your acknowledgements section:

> Music by Kevin MacLeod (incompetech.com), used under CC BY 4.0.

Each track also has its own `<song>.LICENCE.txt` next to the file with the same information.

## Variety for therapy

Tracks intentionally span a range:

- **Gentle (60-90 BPM)**: Bright_Wish, Local_Forecast, Pixelland, good for early rehab, slow purposeful presses.
- **Standard (100-120 BPM)**: most of the rest, typical session pace. Funkorama and Volatile_Reaction have the strongest backbeats for rhythm-mode work.
- **Brisk (130-150 BPM)**: Faster_Does_It and 8bit_Dungeon_Boss, challenge / endurance work.

The difficulty selector (Easy / Medium / Hard) further controls how many beats fire (every 4th, every 2nd, every beat) so the same track can be used at different intensities.

## Adding your own tracks

Drop any `.mp3` / `.wav` / `.ogg` / `.flac` file into this folder. Then either:
- Restart the app and pick it from the rhythm setup screen, or
- Click **Rescan music dir** on the rhythm setup screen.

If you add a track from anywhere other than incompetech.com, save its licence text alongside as `<song>.LICENCE.txt`. Only use CC0, CC BY, or public-domain material.
